### 比较运算符

| 比较运算符          | 功能                                      |
| ------------------- | ----------------------------------------- |
| >                   | 大于                                      |
| >=                  | 大于等于                                  |
| <                   | 小于                                      |
| <=                  | 小于等于                                  |
| =                   | 等于                                      |
| <> 或 !=            | 不等于                                    |
| between ... and ... | 在某个范围之内(含最小、最大值)            |
| in(...)             | 在 in 之后的列表中的值，多选一            |
| like 占位符         | 模糊匹配(\_匹配单个字符, %匹配任意个字符) |
| is null             | 是 null                                   |


### 逻辑运算符

| 逻辑运算符 | 功能                    |
| ---------- | ----------------------- | --- | --------------------------- |
| and 或 &&  | 并且 (多个条件同时成立) |
| or 或      |                         |     | 或者 (多个条件任意一个成立) |
| not 或 !   | 非 , 不是               |
