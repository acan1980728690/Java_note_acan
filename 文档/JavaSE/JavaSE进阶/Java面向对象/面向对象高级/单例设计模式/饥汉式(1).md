# 饿汉可能创建了对象但是没有使用，加载类的时候就会创建对象
实现步骤
- 构造器私有化
- 类的内部创建对象
- 向外暴露一个静态的公共方法
代码实现
```
class SingleTon{
    private SingleTon(){
        //private constructor
    }   
    private static final SingleTon instance = new SingleTon();
    public static SingleTon getInstance(){
        return instance;
    }
}
```
