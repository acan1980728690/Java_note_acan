# 加载类的时候不会创建对象，只有需要对象的时候才会创建
实现步骤
- 构造器私有化
- 声明一个static静态属性对象
- 提供一个public的static方法，可以返回一个Cat对象

**例子**
```
class Singleton {
    private static Singleton instance = null;

    private Singleton() {
        //private constructor
    }

    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}
```
